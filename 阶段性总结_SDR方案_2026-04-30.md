## 3. SDR 测长方法简述

当前 SDR 方法使用 AD9361 2T2R：

```text
DUT 路径: TX1A -> 被测线缆/夹具 -> RX1A
REF 路径: TX2A -> 参考 SMA 回环 -> RX2A
```

程序发送多频点复数 multitone，分别从 RX1/RX2 提取各频点相位，计算 DUT/REF 的复数比值，对相位-频率曲线做线性拟合：

```text
phase(f) ≈ -2πfτ + φ0
length = τ * VF * c
```

当前参数：

```text
sample rate = 30.72 MSPS
LO = 400 MHz
tones = 0.5 MHz ~ 14 MHz, 55 tones
N = 131072
N_AVG = 16
VF_CABLE = 0.66
zero offset = 0.055 m
```

## 4. 通道与 SMA 线排查结论

### A 组：通道交换

A1 正常双回环：

```text
TX1A -> RX1A
TX2A -> RX2A

mean signed length = 0.050 m
span = 0.008 m
valid tones = 55
max residual = 0.13 deg
verdict = OK
```

结论：正常 2T2R 拓扑稳定，通道固定偏置很小。

A2 交叉回环：

```text
TX1A -> RX2A
TX2A -> RX1A

dut_min ≈ -49 dB ~ -50 dB
ref_min ≈ -49 dB ~ -52 dB
valid tones = 0
verdict = LOW_SIGNAL
```

结论：交叉拓扑不可用于校准。A2 反映的是交叉连接下两路都没有有效信号，不是普通通道延迟差。

### B 组：SMA 线交换

B1：

```text
SMA1: TX1A -> RX1A
SMA2: TX2A -> RX2A
mean signed length = 0.057 m
span = 0.007 m
```

B2：

```text
SMA2: TX1A -> RX1A
SMA1: TX2A -> RX2A
mean signed length = 0.053 m
span = 0.002 m
```

由此估算：

```text
系统零点偏置 ≈ (0.057 + 0.053) / 2 = 0.055 m
SMA 线差异 ≈ (0.057 - 0.053) / 2 = 0.002 m
```

结论：SMA1/SMA2 的长度或接头差异不是主要误差源。后续测量使用 `zero_offset_m = 0.055`。

## 5. 线长实验结果

### 零点复测

在正常回环下，使用 `0.055 m` 零偏校正后：

```text
corrected signed len = 0.009 m
corrected span = 0.004 m
valid tones = 55
max residual = 0.17 deg
verdict = OK
```

结论：零点校正有效，SDR 主链路稳定。

### 1.12 m 标识短线

最稳定一组结果：

```text
forward:
corrected signed len = 1.089 m
corrected span = 0.002 m
max residual = 0.99 deg

reversed:
corrected signed len = 1.087 m
corrected span = 0.005 m
max residual = 0.97 deg
```

对 1.12 m 的相对误差约为：

```text
(1.089 - 1.12) / 1.12 = -2.8%
(1.087 - 1.12) / 1.12 = -2.9%
```

另有一次单独测试的校正结果约为 `1.117 m`，非常接近 1.12 m。

结论：1.12 m 短线在夹具状态稳定时可以获得较好结果；但短线对 RJ45/SMA 夹具接触状态敏感，不能把所有短线数据都当作精度标定依据。

### 1.10 m 标识短线

用户确认 1.10 m 线物理上短于 1.12 m。但多组 SDR 测量出现：

```text
1.470 m / 1.456 m
1.519 m / 1.517 m
1.617 m / 1.581 m
```

这些结果与物理事实冲突。对应 residual 多在 `4 deg ~ 5 deg`，说明相位-频率拟合不够线性。

结论：当前临时 RJ45/SMA 夹具不能稳定分辨 1.10 m 与 1.12 m。论文中不要把“分辨 2 cm 差异”作为成果目标。

### 约 7.91 m 长线

长线物理估计：

```text
4 * 1.10 m + 3 * 1.12 m + 0.15 m = 7.91 m
```

20 次长线测量汇总：

```text
corrected length mean = 8.147 m
overall span = 0.094 m
max residual = 2.49 deg
valid tones = 55
error vs 7.91 m estimate = +2.99%
```

分组结果：

```text
正接 forward  = 8.167 m
正接 reversed = 8.186 m
反接 forward  = 8.104 m
反接 reversed = 8.130 m
```

结论：长线结果稳定且误差约 3%，比短线更能代表电赛基本要求的应用场景。该结果可以作为论文 SDR 测长模块的主要实验成果。

## 6. 当前质量门限建议

后续软件或论文分析中建议采用以下判据：

```text
valid tones = 55
residual <= 3 deg
repeat span <= 0.10 m
```

解释：

```text
长线实验 residual 最大为 2.49 deg，span 为 0.094 m，可以通过。
1.10 m 异常短线 residual 多为 4~5 deg，应标记为可疑，不作为有效精度样本。
LOW_SIGNAL、FEW_VALID_TONES、HIGH_RESIDUAL、UNSTABLE 均不用于校准。
```

## 8. 重要结论

目前可以确定：

1. AD9361 2T2R 相位斜率测长方案是可行的。
2. 正常拓扑 `TX1A -> RX1A` 和 `TX2A -> RX2A` 稳定。
3. 交叉拓扑 A2 不可用于通道校准。
4. SMA 线差异约 `2 mm` 等效长度，可忽略。
5. 系统零偏可取 `0.055 m`。
6. 1.12 m 短线最好结果约 `1.09 m ~ 1.12 m`。
7. 1.10 m 与 1.12 m 的 2 cm 差异目前不能稳定分辨。
8. 约 7.91 m 长线平均测得 `8.147 m`，误差约 `3%`，可作为主要成果。
9. 当前主要误差来源是 RJ45/SMA 临时夹具、回流路径和接触状态，不是 SDR 主链路。
